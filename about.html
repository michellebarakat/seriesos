About SeriesOS

A personal TV tracker and discovery site built for the love of binge-watching.

What is SeriesOS?

SeriesOS is a personal watchlist and discovery platform. It has two parts: a hand-curated collection of 29 shows I've personally watched and reviewed, and a Discover page powered by the TMDB API where you can search any show or movie in existence. Click any card to open a detailed view with the backdrop, synopsis, rating, and my personal take.

Custom UI Feature

The unique front-end requirement for this project was implementing a modal popup for a detailed item view. Every card in both the Collection and Discover pages opens a full modal showing the backdrop image, poster, title, rating, TMDB overview, and (on the collection page) my own personal review. The modal closes on backdrop click, the × button, or the Escape key — all handled via Bootstrap 5's modal component wired up through ES6 class methods.
